About the Project

What is Project Monster Killer?

Monster Killer is a browser-based game project made with HTML5 Canvas element and Vanilla Javascript.

How does it work?

The objective of the game is to kill as many monsters in 20 seconds.

Why did you choose it?

WAfter seeing many examples I decided to make a simple game, but at the same time I could apply new knowledge not only on canvas, but especially Javascript.
